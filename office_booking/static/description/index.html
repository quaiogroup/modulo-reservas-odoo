Office Booking

Complete office & coworking space management for Odoo 19

Key Features

Space Management

  • Manage multiple offices and coworking spaces
  • Define availability blocks and time slots
  • Upload multiple images per space
  • Associate services and amenities

Online Booking

  • Public website booking page with live calendar
  • Real-time slot availability check
  • Hourly and daily booking options
  • Instant booking confirmation emails

Coworking Plans & Subscriptions

  • Create flexible membership plans (hourly, daily, monthly)
  • Manage active subscriptions per client
  • Automatic billing cycle tracking
  • Plan-based discounts applied at booking

Discount Codes

  • Create percentage or fixed-amount discount codes
  • Set validity dates and usage limits
  • Apply codes at checkout on the website
  • Track usage count per code

Payment Integration

  • Bold payment gateway support
  • Configurable API key via system parameters
  • Automatic booking confirmation on payment
  • Payment status tracking per booking

Email Notifications

  • Booking confirmation email to client
  • Internal notification to admin on new booking
  • Cancellation notification emails
  • ICS calendar attachment for confirmed bookings

Customer Portal

  • Full booking history in customer portal
  • Booking detail page with status and actions
  • Download booking receipt as PDF
  • Add to Google Calendar / iCal links
  • Cancel booking from portal

Analytics Dashboard

  • Revenue and occupancy analytics
  • Booking trends by space and period
  • Availability overview dashboard
  • Export bookings to CSV/PDF

Configuration

After installation, go to Office Booking > Configuration > Settings to configure your payment gateway API key and admin notification email. All sensitive values are stored as Odoo system parameters — never hardcoded.
